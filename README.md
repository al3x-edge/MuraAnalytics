#MuraAnalytics

This plugin allows the addition of Google Analytics tracking codes. This has been updated for the new asynchronous ga.js as well as the analytics.js for Universal Analytics.

## Authors
Alexander Ambrose (fh group) - al3xnull

##Tested With
* Mura CMS Core Version 6.1
* Railo 4.0.4
